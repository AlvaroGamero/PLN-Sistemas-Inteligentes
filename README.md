# PLN-Sistemas-Inteligentes

Primera entrega

🛠️ Crea virtual env: `virtualenv venv`

* Activalo en windows: `C:\> env\Scripts\activate.bat`

* Activalo en linux:`pip install -r requirements.txt`

🚀 Ejecuta el programa: `python app.py`

🏗 Estructura:

```
.
├── app.py
├── README.md
├── requirements.txt
├── src
│   └── analisis.py
├── tweets.xlsx
└── venv
    ├── bin
    ├── lib
    └── pyvenv.cfg
```
